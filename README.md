# dwtest

Durbin-Watson Test for Python. This module is the python-version for the `dwtest` function of `lmtest` library in R Language.

## Install

The quick way:

```
pip install dwtest
```

## Usage

Simple to use:

```python
from dwtest import dwtest

statistic, pvalue = dwtest(formula, data)
```
